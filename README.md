# Project Horizon - Learning and Skill Development Platform

## 📚 Overview
Project Horizon is a modern web-based Learning and Skill Development Platform designed to provide students, professionals, and learners with a centralized digital environment for acquiring knowledge and developing new skills.

## 🎯 Key Features

### Student Module
- User Registration and Login
- Course Enrollment
- Access Learning Materials
- Watch Video Lectures
- Take Online Quizzes
- View Progress Reports
- Download Certificates

### Instructor Module
- Create and Manage Courses
- Upload Learning Content
- Create Quizzes and Assignments
- Monitor Student Performance

### Admin Module
- User Management
- Course Management
- Analytics Dashboard
- Content Approval and Monitoring

## 🤖 AI Features
1. AI Course Recommendation
2. AI Learning Assistant (Chatbot)
3. AI Quiz Generator
4. AI Resume & Skill Analyzer

## 💻 Technology Stack
- React.js, Tailwind CSS, JavaScript
- Node.js, Express.js, MongoDB
- JWT Authentication, Cloudinary

## 📦 Quick Start

### Backend
```bash
cd backend
npm install
cp .env.example .env
npm start
```

### Frontend
```bash
cd frontend
npm install
cp .env.example .env
npm start
```
